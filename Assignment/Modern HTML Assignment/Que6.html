Differences between HTML and HTML5

HTML HTML5
HTML stands for HyperText Markup Language. HTML5 stands for HyperText Markup Language version 5.
HTML4 was the previous version of HTML. HTML5 is the latest and current version of HTML.
HTML4 had limited support for multimedia and interactive elements. HTML5 introduced native support for audio, video, and canvas elements, making it suitable for multimedia-rich web applications.
